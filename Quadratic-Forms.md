# Quadratic Forms(二次型)

**Definition 7.2.3.**  If $`A`$ is a symmetric $`m \times m`$ matrix, the **quadratic form** defined <br>
by $`A`$ is the function $`q_{A}(x) = x \cdot (Ax) `$. <sup>[1]</sup>
