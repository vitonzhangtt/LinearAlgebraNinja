# Bases

## Definition
Definition 3.2.3.  A set of vectors $`\mathbf{v}_{1}, v_{2}, ..., v_{n}`$ in $`\mathbb{R}^m`$ <br>
is called a **basis** for $`\mathbb{R}^m`$ if the set of vectors **spans** $`\mathbb{R}^m`$ and  <br>
is linearly independent. <sup>[1]</sup>
