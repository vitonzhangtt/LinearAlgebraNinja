## Null Space

The **null space** of a matrix consists of **all the vectors** that get mapped to the zero vector <br>
when multiplied by that matrix. <sup>[1]</sup>

Suppose we have a matrix $`A`$, the null space of matrix $`A`$ is a set of all possible vectors $`X`$ <br>
such that the equation $`AX=0`$ is satisfied. <sup>[1]</sup>

## Row Space

It(Row Space) is a set of linear combinations of the row vectors of a matrix. <sup>[1]</sup>



## Column Space

The column space of a matrix ($`A`$) is the subspace spanned by its column vectors. In other words, <br>
it’s the set of **all linear combinations** of these columns. <sup>[1]</sup>

When you multiply a vector $`X`$ by a vector $`A`$, the **resulting vector** is **in** the column space of $`A`$.

For example, <br>

A is a matrix: $`A = [v_{1}, v_{2}, v_{3}]`$, $`v_{1}, v_{2}, v_{3}`$ are columns of matrix $`A`$. <br> 
$`X`$ is a vector, $`X^T = [x_{1}, x_{2}, x_{3}]`$.

$`AX = [v_{1}, v_{2}, v_{3}]X = v_{1}x_{1} + v_{2}x_{2} + v_{3}x_{3}`$
